# wordle-it-solver

Solver for [wordle-it](https://github.com/pietroppeter/wordle-it) by @pietroppeter

### How to run
```
virtualenv -p python3 venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 solver.py
```
